ERP系统操作指南

一、系统登录

  1. 打开浏览器，输入 ERP 系统地址：http://erp.company.com
  2. 输入用户名和密码。
  3. 首次登录请修改默认密码，并绑定邮箱与手机号码。
  4. 登录后进入“工作台”，可以看到今日任务和待审批事项。

二、采购管理

  • 创建采购申请：

    1. 进入“采购管理” → “采购申请” → 点击“新建”。
    2. 选择供应商和物料。
    3. 填写数量、交货期、预算等信息。
    4. 提交审批后，系统会自动生成采购订单。

  • 采购订单审核：

    1. 审核角色登录“采购管理” → “采购订单审批”。
    2. 点击订单 → 查看详情 → 点击“通过”或“驳回”。
    3. 系统会通知申请人审批结果。

三、库存管理

  1. 库存查询：
    • 进入“库存管理” → “库存查询”，输入物料编码即可查询。
  2. 库存盘点：
    • 点击“盘点任务” → 选择仓库 → 录入实物数量 → 提交。
  3. 库存预警：
    • 系统会自动发送库存不足提醒邮件给采购人员。

四、财务管理

  • 发票录入：
    1. 进入“财务管理” → “发票管理” → “录入发票”。
    2. 上传发票扫描件，填写金额、供应商信息。
    3. 系统自动匹配付款计划。
  • 付款申请：
    1. 进入“付款管理” → “新建付款申请”。
    2. 选择审批流程 → 提交 → 财务审核 → 执行付款。

五、常见问题

  1. 忘记密码怎么办？
    • 点击登录页“忘记密码”，按照提示重置。
  2. 审批流程卡住？
    • 检查流程节点是否有人未审批，或者联系系统管理员。
  3. 导出报表乱码？
    • 确认浏览器编码为 UTF-8，或尝试下载 Excel 格式报表。

六、升级说明

  • 系统升级时间：每月第 3 个周日凌晨 1:00~5:00
  • 升级内容：
    1. 优化采购审批流程，提高审批效率。
    2. 新增库存预警邮件通知功能。
    3. 修复已知报表导出乱码问题。
  • 升级注意事项：
    1. 升级期间请勿登录系统操作。
    2. 升级完成后清理浏览器缓存。

✅ 说明：

结构清晰：登录、采购、库存、财务、常见问题、升级说明
